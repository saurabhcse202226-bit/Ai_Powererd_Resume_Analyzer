About ResumeAI

ResumeAI is a final year project built to help students and job seekers understand exactly how their resume stacks up against job requirements.

It uses NLP techniques to extract skills, experience, and education from uploaded resumes and compares them against industry job profiles. Claude AI then provides personalized feedback and actionable improvement suggestions.

Tech Stack
  • Python + Flask (Backend)
  • SQLite + SQLAlchemy (Database)
  • pdfplumber + python-docx (Parsing)
  • Claude API (AI feedback)
  • Bootstrap 5 (Frontend)
Developer

Built as a portfolio/internship project. The goal was to create something actually useful while learning Flask, NLP, and AI APIs.
